# Contributing

This project is currently under active development and licensing review.

At this stage:

- external contributions are not yet being formally accepted
- bug reports, design observations, and practical feedback are welcome
- the codebase, repository layout, and licensing model are still being stabilized

## Before Contributing

Please understand that:

- command behavior and internal structure may change
- some parts of the repository are still being cleaned and reorganized
- the final contribution and licensing policy has not yet been locked down

## Feedback

For now, the most useful forms of contribution are:

- issue reports
- reproducible bug notes
- build observations
- documentation corrections
- design feedback

## Code Contributions

Please do not submit pull requests yet unless invited to do so.

Formal contribution guidelines will be added after:

- licensing is finalized
- repository structure is stabilized
- public development workflow is defined
