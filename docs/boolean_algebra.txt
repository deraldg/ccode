Here’s a compact reference of **Boolean operator symbols**, useful across programming, logic, and search contexts:

### 🧠 Core Boolean Operators

| Operator | Symbol(s)     | Meaning                        | Example Usage                    |
|----------|---------------|--------------------------------|----------------------------------|
| AND      | `AND`, `&&`, `&` | True if both conditions are true | `A && B` or `A AND B`            |
| OR       | `OR`, `||`, `|`  | True if at least one is true     | `A || B` or `A OR B`             |
| NOT      | `NOT`, `!`, `¬`  | Inverts the truth value          | `!A` or `NOT A`                  |
| XOR      | `^`              | True if only one is true         | `A ^ B`                          |

### 🔍 Search & Query Enhancements

| Symbol      | Function                                  | Example                          |
|-------------|-------------------------------------------|----------------------------------|
| `"`         | Exact phrase match                        | `"climate change"`               |
| `*`         | Wildcard for variations                   | `develop*` → development, developer |
| `()`        | Grouping terms                            | `(solar OR wind) AND energy`     |
| `AND NOT` / `-` | Exclude terms                        | `apple AND NOT fruit` or `apple -fruit` |

### 🧮 Formal Logic Symbols

| Symbol | Name               | Meaning                        |
|--------|--------------------|--------------------------------|
| `∧`    | Logical AND         | True if both operands are true |
| `∨`    | Logical OR          | True if at least one is true   |
| `¬`    | Negation            | Inverts truth value            |
| `⇒`    | Implication         | If A then B                    |
| `⇔`    | Biconditional       | A if and only if B             |
