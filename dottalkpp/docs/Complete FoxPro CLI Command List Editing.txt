#### Key Database-Related Commands (Prioritized for Your Library)
These align with your library's functions (e.g., use(), build_index(), etc.).

- **USE**: Syntax: `USE <table_name> [IN <area>] [INDEX <index_file>]` - Opens a table (DBF file) in a specified work area, optionally with an index.
- **CREATE**: Syntax: `CREATE <table_name>` - Creates a new table (DBF file).
- **INDEX**: Syntax: `INDEX ON <expression> TAG <tag_name> [OF <index_file>]` - Creates an index on a table.
- **REINDEX**: Syntax: `REINDEX` - Rebuilds all indexes for the current table.
- **SELECT**: Syntax: `SELECT <area_number>` - Switches to a specific work area.
- **CLOSE**: Syntax: `CLOSE [TABLES | INDEXES | DATABASES]` - Closes open tables, indexes, or databases.
- **APPEND BLANK**: Syntax: `APPEND BLANK` - Adds a blank record to the current table.
- **DELETE**: Syntax: `DELETE [FOR <condition>]` - Marks records for deletion.
- **REPLACE**: Syntax: `REPLACE <field> WITH <value> [FOR <condition>]` - Replaces field values in records.
- **LOCATE**: Syntax: `LOCATE FOR <condition>` - Searches for the first record matching a condition.
- **FIND**: Syntax: `FIND <key_value>` - Searches an indexed table for a key (requires macro expansion with '&' in some contexts).
- **SKIP**: Syntax: `SKIP [<n>]` - Moves the record pointer forward or backward by n records.
- **GO TOP**: Syntax: `GO TOP` - Moves to the first record.
- **GO BOTTOM**: Syntax: `GO BOTTOM` - Moves to the last record.
- **SET ORDER**: Syntax: `SET ORDER TO <tag_name>` - Sets the controlling index tag.
- **SET RELATION**: Syntax: `SET RELATION TO <expression> INTO <table>` - Links tables based on a common field.
- **COPY TO**: Syntax: `COPY TO <new_table> [FOR <condition>]` - Copies records to a new table.
- **ZAP**: Syntax: `ZAP` - Permanently removes all records from a table.
- **APPEND FROM**: Syntax: `APPEND FROM <source_table>` - Appends records from another table.
- **SORT**: Syntax: `SORT TO <new_table> ON <field> [/A|/D]` - Sorts the table and creates a new one.
- **COUNT**: Syntax: `COUNT [FOR <condition>]` - Counts records matching a condition.
- **SET DELETE**: Syntax: `SET DELETE ON|OFF` - Toggles display of deleted records.
- **SET TALK**: Syntax: `SET TALK ON|OFF` - Toggles command result display.

#### Startup Command-Line Options (Not CLI Verbs, but Related)
These are for launching FoxPro/VFP.
- -A: Ignore default config.
- -B<FileName,Duration>: Display custom bitmap.
- -C<FileName>: Specify config file.
- -L<FileName>: Specify resource file.
- -R: Refresh registry (older versions).
- -T: Suppress splash screen.
- /?: List arguments.
- /regserver: Register keys.
- REGSVR32 <dll>: Register DLL.
- /unregserver: Unregister keys.
- /u <dll>: Unregister DLL.

        token = strtok(NULL, " ");
