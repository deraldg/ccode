# The Correct Fix

You need **one canonical execution function** inside the shell core.

Example:

```cpp
bool shell_execute_line(xbase::DbArea& area, const std::string& raw)
{
    std::string line = textio::trim(raw);
    if (line.empty())
        return true;

    // handle SET/SHOW/CLEAR VAR
    {
        auto vr = dottalk::try_handle_var_command(area, line);
        if (vr.handled)
            return vr.ok;
    }

    // expand &macros
    {
        std::string expanded;
        std::string err;

        if (!dottalk::expand_macros_outside_quotes(line, expanded, err)) {
            std::cout << "Undefined VAR '" << err << "'\n";
            return false;
        }

        line = std::move(expanded);
    }

    // IF suppression logic
    if (shell_if_is_suppressed())
        return true;

    // final dispatch
    return shell_dispatch_line(area, line);
}
```

---

# Then route everything through it

### interactive prompt (`shell.cpp`)

Replace:

```cpp

```

with

shell_execute_line(area, line);
