COUNT parity with LOCATE + predicate engine; SETCASE + UPPER/LOWER()

- New COUNT implementation using shared predicate evaluator (same as LOCATE)
- Supports: ALL | DELETED | FOR <field|UPPER/LOWER(field)> <op> <value>
- Case control via SETCASE ON|OFF (default OFF)
- Inline comments '&&' stripped from COUNT/LOCATE lines
- Added PREDHELP command with operator/case/function docs

Verified:
- LAST_NAME $ "MIT" -> 8
- GPA = 3.43 -> 2
- IS_ACTIVE = .T. -> 42
- UPPER/LOWER() work in both COUNT and LOCATE
